Project Description:
This project is a web-based Expense Tracker application that allows users to record their daily expenses and manage financial data efficiently.

The system provides multiple features such as adding expenses, viewing stored data, and generating reports.

----------------------------------------

Features:
- User-friendly interface
- Add expense details (date, category, amount, description)
- View expense report in table format
- Automatic total expense calculation
- Data stored using browser localStorage

----------------------------------------

Technologies Used:
- HTML
- CSS
- JavaScript

----------------------------------------

Conclusion:
This project helps users track their daily spending and provides a simple way to manage expenses effectively.
